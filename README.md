# Puzzle Generator

Puzzle Generator is a web-based application offering two interactive logic games: **Sudoku** and **Word Search**. The project focuses on clean design and smooth animations.

## Features
- **Sudoku**
  - Auto-generated puzzles with multiple difficulty levels  
  - Smart input, mistake highlighting, hints, and auto-detect completion  
  - Confetti celebration + automatic new puzzle on completion  
  - Mobile-friendly number pad  

- **Word Search**
  - Always-solvable generated grids  
  - Smooth drag-to-highlight interaction  
  - Dynamic highlight lines that adapt to screen resizing  
  - Automatic completion detection with confetti
